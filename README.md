# 🛵 外送收入追蹤器

> 專為外送員設計的 PWA 應用程式，記錄每日收入並分析最賺錢的時段。

## ✨ 功能

- **新增每日紀錄** — 輸入上線時間、總收入、完成單數
- **自動計算** — 時薪、每單均價、上線時數即時顯示
- **週報表** — 以週為單位瀏覽，可切換前後週
- **智慧分析** — 自動標示本週哪天最賺、哪天最不划算（依時薪排名）
- **視覺化圖表** — 切換顯示收入 / 時薪 / 均價 / 單數長條圖
- **離線可用** — Service Worker 快取，無網路也能使用
- **安裝到主畫面** — 支援 PWA 安裝，用起來像 App

## 🚀 使用方式

### 直接用 GitHub Pages

1. Fork 這個 repo
2. 到 Settings → Pages → Source 選 `main` branch
3. 開啟 `https://你的帳號.github.io/delivery-tracker/`

### 本地使用

下載 ZIP 後，用瀏覽器直接開啟 `index.html` 即可（部分 PWA 功能需 HTTPS）。

## 📁 檔案結構

```
delivery-tracker/
├── index.html      # 主程式（單一檔案，包含所有 HTML/CSS/JS）
├── manifest.json   # PWA 設定檔
├── sw.js           # Service Worker（離線快取）
└── icons/
    ├── icon-192.png
    └── icon-512.png
```

## 📊 資料欄位

每筆紀錄包含：

| 欄位 | 說明 |
|------|------|
| 日期 | 外送日期 |
| 開始時間 | 上線時間 |
| 結束時間 | 下線時間 |
| 總收入 | 當日總金額（元） |
| 完成單數 | 當日總外送單數 |
| 備註 | 天氣、活動加成等備注 |

自動計算欄位：**時薪**、**每單均價**、**上線時數**

## 💾 資料儲存

所有資料存在瀏覽器的 `localStorage`，不會上傳任何伺服器，完全隱私。

> ⚠️ 清除瀏覽器資料會刪除紀錄，建議定期截圖備份。
